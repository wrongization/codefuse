# 环境配置和使用


~~~cmd
后端参数设置（根据实际情况修改）：
backend\.env
前后端依赖安装：
set-env.bat
后端py环境依赖配置（需要uv）：
backend\set_env.bat
前端node包配置：
frontend\set_env.bat
启动前后端：
start-all.bat
关闭前后端：
close-all.bat
启动前端：
frontend\start.bat
关闭前端：
frontend\close.bat
启动后端：
backend\start.bat
关闭后端：
backend\close.bat
重置数据库（会清空数据）：
reset_database.bat
~~~

前端地址：http://localhost:5173/
